Setup and Usage

Matlab Wrapper

After installing the FAuST package, it’s straight forward, in a terminal type:

    $ cd /opt/local/faust-*/matlab
    $ matlab -nodisplay

Here you are in Matlab terminal. Note that you can also do it through Matlab GUI.

The following script will add FAuST in your PATH. This is needed for Matlab to use FAuST.

    >> setup_FAUST

Now let’s test FAuST with the quickstart script:

    >> quick_start

And then type further instructions to test a bit of FAuST API:

Let’s see what variables quickstart script has added.

    >> whos

Now call some functions on FAuST object A:

    >> RCG(A)
    >> density(A)
    >> get_nb_factor(A)

Retrieve the product factors (as full matrices):

    >> F1 = get_fact(A,1);
    >> F2 = get_fact(A,2);

Check the sizes:

    >> size(F1)
    >> size(F2)
    >> size(A)

For more information on FAuST API, and whole function listing, consult the doc:

    >> doc('Faust')

N.B.: to access the documentation, you need to be in the graphical interface of matlab.

Python Wrapper

In the same idea, the python wrapper of FAuST is usable going into its directory.

    $ cd /opt/local/faust-*/python
    $ python2.7 quickstart.py

You can also go through python terminal to build a FAuST product and call its object methods.

    $ python2.7
    >>> import FaustPy
    >>> import FaustPy
    >>> A = FaustPy.Faust('A.mat') # A is the FAuST created through quickstart script
    >>> A.RCG()
    6.666666666666667
    >>> A.density()
    0.15
    >>> A.get_nb_factors()
    2
    >>> F1 = A.get_factor(0)
    >>> F2 = A.get_factor(1)
    >>> A.size()
    (1000, 2000)
